Rules


This typing game tests the speed and accuracy you can type different phrases.


To play the game:


  1. Press the start game button
  2. A random phrase will be displayed in the game box
  3. Type the phrase into the input field below
  4. On completion of a phrase, press the enter key. A new phrase will be displayed
  5. You have two minutes to type as many phrases as you can
  6. When the timer runs out, the game will end
  7. To start a new game, simply press the press the start game button
